<h1>🎬 MovieZilla</h1>

<p>
<b>MovieZilla</b> is a modern movie discovery web application that allows users to explore trending films, browse categories, search for movies, and view detailed information including cast, ratings, and related recommendations.
</p>

<p>
The project focuses on building a <b>clean, responsive, and performant UI</b> while implementing real-world frontend architecture using <b>React</b>.
</p>

<hr>

<h2>✨ Features</h2>

<ul>
<li>🔎 <b>Search Movies</b> – Instantly search movies using a dynamic search system.</li>
<li>🎭 <b>Movie Details Page</b> – View full movie information including overview, ratings, cast, and poster.</li>
<li>🎞 <b>Category Browsing</b> – Explore movies by categories like trending, popular, and top rated.</li>
<li>🎯 <b>Related Movie Suggestions</b> – Displays recommended movies based on the selected film.</li>
<li>📱 <b>Responsive Design</b> – Optimized layout for desktop, tablet, and mobile devices.</li>
<li>⚡ <b>Fast UI Rendering</b> – Optimized React component structure for smooth interactions.</li>
<li>🎨 <b>Modern UI</b> – Clean card-based layout with hover interactions and smooth transitions.</li>
<li>🔗 <b>Dynamic Routing</b> – Each movie has a dedicated route with detailed information.</li>
</ul>

<hr>

<h2>🧠 What I Focused On</h2>

<ul>
<li><b>Component-based architecture</b> for reusable UI elements.</li>
<li><b>API data fetching and handling</b> to dynamically render movie data.</li>
<li><b>State management</b> to control movie lists, search results, and UI updates.</li>
<li><b>Responsive UI design</b> using modern CSS techniques.</li>
<li><b>Clean folder structure</b> for scalable frontend development.</li>
<li><b>User experience improvements</b> such as hover effects, smooth transitions, and intuitive navigation.</li>
</ul>

<hr>

<h2>🛠 Tech Stack</h2>

<ul>
<li><b>React</b> – Frontend framework</li>
<li><b>JavaScript (ES6+)</b> – Core programming language</li>
<li><b>CSS / SCSS</b> – Styling and layout</li>
<li><b>REST API</b> – Movie data integration</li>
<li><b>React Router</b> – Client-side routing</li>
</ul>
